This section presents how to configure the dependencies used by the application. Below, you will find those required for the system to run, as well as optional ones depending on your needs.

Optional dependencies are like plugins that can extend the functionality of OpenGateLLM. For example, enabling Elasticsearch activates endpoints related to RAG.
