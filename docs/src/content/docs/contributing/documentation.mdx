<Aside type="caution" title="Prerequisites" icon="approve-check-circle">
        - Node.js (version 18+ recommended)
        - npm (or yarn/pnpm)
</Aside>

OpenGateLLM uses [Astro Starlight](https://starlight.astro.build/) to serve its documentation. Start the local development server with the following command:

<Steps>
1. Go to the documentation folder

   ```bash
   cd docs
   ```

2. Install the dependencies

   ```bash
   npm install
   ```

3. Run the development server

   ``` bash
   npm run dev
   ```

   This will:
   -   Install the dependencies
   -   Launch a development server at **http://localhost:4321/**
   -   Hot-reload changes as you edit files

4. Edit the documentation


   <FileTree>
   - astro.config.mjs Astro configuration file
   - src
      - assets Assets folder
        - icons/ Custom icons SVG folder
     - content
       - **docs/** Documentation markdown files folder
   </FileTree>
</Steps>

<Aside type="danger" title="Automatic documentation generation">
Some documentation is automatically generated from the codebase. Please do not edit the generated documentation files:
- [Configuration file](/configuration/configuration_file/) (see [generation script](https://github.com/etalab-ia/OpenGateLLM/blob/main/scripts/docs/generate_configuration_documentation.py))
</Aside>

## Create terminal GIF

<Steps>
1. Install asciinema ((see documentation)[https://docs.asciinema.org/getting-started/])

2. Create a recording of the terminal command

   ```bash
   # example for make quickstart command
   asciinema rec docs/src/assets/make_quickstart.cast \
   --overwrite \
   --command "make quickstart" \
   --title "make quickstart" \
   --window-size 120x24
   ```

3. Build asciinema gif generator (agg) image docker to convert the recording to a GIF (see [asciinema/agg](https://github.com/asciinema/agg))

4. Convert the recording to a GIF

   ```bash
   docker run --rm -it -u $(id -u):$(id -g) \
   -v $PWD:/data agg \
   /data/docs/src/assets/make_quickstart.cast \
   /data/docs/src/assets/make_quickstart.gif
   ```
</Steps>
